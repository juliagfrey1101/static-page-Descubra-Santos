# 🌊 Descubra Santos - Guia Turístico

Este é um projeto de uma **Landing Page** moderna e responsiva focada no turismo da cidade de Santos. O objetivo foi criar uma interface limpa, elegante e funcional utilizando apenas tecnologias fundamentais do desenvolvimento Web.

## 🚀 Sobre o Projeto
A página foi desenhada para oferecer uma experiência de utilizador agradável, com foco em tipografia refinada e imagens de alto impacto. O site apresenta secções de História, Praias, Atrações, Restaurantes e Hospedagem.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização avançada, incluindo:
  - Flexbox/Grid para layout.
  - Google Fonts (tipografia personalizada).
  - Efeitos de transparência e overlays em imagens.
  - Design Responsivo (adaptável a diferentes ecrãs).

## 🎨 Destaques do Design
- **Header Minimalista:** Navegação intuitiva com ícones de Favoritos e Perfil.
- **Hero Section:** Imagem de fundo com overlay escuro para destacar a frase de impacto e o botão de Call to Action (Guia e Pacotes).
- **Paleta de Cores:** Focada no contraste entre o preto, branco e tons de cinza para uma estética "Premium".
